Buscando elementos

Para encontrar um elemento basta usar "document.querySelector(elemento)"

O document serve para selecionar o documento HTML, e o querySelector ira buscar o elemento

Exemplo pratico:

Aqui temos 3 botões e ao serem clicados soltam um som ou alerta

function tocaLatido() {
    document.querySelector('#som_latido').play();
}
document.querySelector('.tecla_latido').onclick = tocaLatido;
